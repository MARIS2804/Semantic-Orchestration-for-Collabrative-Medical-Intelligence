###  SOCMI: Semantic Orchestration for Collaborative Medical Intelligence

This project implements **SOCMI**, a Hierarchical Multi-Agent System (HMAS) designed for high-precision clinical diagnosis using Electronic Medical Records (EMRs). It merges the sophisticated task-coordination logic of the **AutoHMA-LLM** framework with the structural knowledge integration of the **medIKAL** paper. The architecture utilizes a dual-tier "Brain" consisting of a **Planner** (Qwen-1.5B) for hypothesis generation and a **Dispatcher** (Phi-3-Mini) for routing complex tasks between specialized agents. These agents include an **NLP Summarizer** for feature extraction, a **RAG Specialist** for searching clinical guidelines in a FAISS vector database, and a **KG Agent** for discovering structural neighbors in a Neo4j Knowledge Graph. A core innovation is the implementation of **medIKAL’s Algorithm 2**, which performs path-based reranking to calculate synergy scores between patient symptoms and candidate diseases. To ensure clinical standardization, the system uses **BioLORD semantic mapping** to resolve raw text into high-dimensional graph nodes. The entire pipeline is **GPU-accelerated** using half-precision (FP16) logic, allowing for complex medical reasoning to occur locally and securely without internet dependency. Ultimately, the system provides a validated **Differential Diagnosis (Top 5)**, where each condition is ranked by evidence strength and explained through a hierarchical reasoning loop.
